# Aibou-Resources
Support files or guides for Aibou Game Translate

## Google Cloud Translate Guide
## Step 1: Create a new Project. Name Can be whatever you want.
![step 1.1](/images/01_project_page.png.webp)
![step 1.2](/images/02_create-new-project.png.webp)
![step 1.3](/images/03_create_project.png.webp)
## Step 2: With the new project selected, go to the hamburger menu to the left and visit API & Services
![step 2.1](/images/04_api_and_services.png.webp)
## From Dashboard press Enable API and Services
![step 2.2](/images/enable_google_translate_api.png.webp)
## Search for "Translate", select Cloud Translate API and enable it
![step 2.3](/images/search_translate_api.png.webp)
![step 2.4](/images/google_cloud_translation_api.png.webp)
## Step 3: Press Credentials from the sidebar and create a new API Key.
![step 3.1](/images/05_create_api_key.jpg)
## Step 4: Press "Show" next to the API key, copy it, and paste it into "Google Translate API Key" from the home page of the app.
![step 4.1](/images/05_the_api_key.jpg)
![step 4.2](/images/05_paste_api_key.jpg)
